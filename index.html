DAY 2 · 내 첫 페이지7 / 26

내 첫 페이지 직접 만들기
레고처럼 블록을 쌓으면 어떤 형태의 노트든 만들 수 있어요
페이지가 뭔가요?

노션에서의 페이지는 하나의 문서예요. 공책의 한 장 같은 거예요.

그 페이지 안에 글도 쓰고, 사진도 넣고, 표도 만들 수 있어요.

페이지가 모이면 사이드바에 목록으로 쭉 보여요.

✏️ 따라해보기 — 첫 페이지 만들기
  1. 왼쪽 사이드바 하단 "+ 새 페이지" 클릭
  2. 빈 화면이 열려요 → 제목 자리에 "나의 첫 노션 페이지" 입력
  3. 제목 아래 빈 공간을 클릭하고 아무 글자나 써보세요
  4. 저장 버튼 따로 없어요. 자동 저장이에요!

여기서 또 '+새 페이지'로 다른 페이지를 만들면? 이렇게 페이지가 하나 둘 쌓여가는 거에요.
지금은 잘 이해가 안되도 괜찮아요.
처음엔 개념을 익히겠지만, 결국에 우린 그걸 정리하고 구조화 할 거에요.

💡 노션 TIP — 제목 왼쪽 빈 영역을 클릭하면 페이지에 이모지 아이콘을 달 수 있어요.
(이모지 덕분에)왼쪽 사이드바에서도 훨씬 쉽게 찾을 수 있어요 (예쁜건 덤)

DAY 3 · 페이지 VS 블록9 / 26
🧱 Day 3
페이지 vs 블록
이거 이해하면 노션의 절반은 이해한 거예요
우리 노션대학 학습지에서는 블록 → 페이지 순으로 따라올게요 😊
이렇게 생각하면 쉬워요

페이지는 공책의 한 장이에요. (문서, 노트, 공책 등등 이라고도 불러요)

블록은 그 장 위에 쓴 글, 붙인 사진, 그린 표 등등 이예요.

노션의 모든 내용은 블록으로 이루어져 있어요. 텍스트 한 줄도 블록, 사진도 블록, 표도 블록, 데이터베이스라고 불리는 것도 블록이에요.

말그대로 다 블록이에요. 그래서 노션을 '레고'에 비유하는 분들도 꽤 있어요.
귀엽지 않나요? 😄

📄
페이지 (Page)
하나의 문서 공간.
왼쪽 사이드바에서 목록으로 보여요.
🧱
블록 (Block)
페이지 안의 모든 요소.
글, 사진, 표, 체크박스 다 블록!
💡 블록의 진짜 힘

블록은 드래그해서 순서를 바꿀 수 있어요. (단축키로도 가능)

'이게 저기로 가면 좋겠는데'라는 생각이 들면, 블록 덩어리채 옮기면 돼요.

블록 왼쪽에 ⠿ 아이콘을 잡고 드래그해보세요!
(단축키 alt+shift+방향키 위/아래키)

DAY 3 · 슬래시 명령어10 / 26
🧱 Day 3
블록 만드는 마법 — 슬래시 /
이거 하나만 알면 웬만한 블록은 다 만들 수 있어요
슬래시가 뭔가요?

노선 페이지에서 키보드의 '/'(슬래시)를 입력하면 팝업처럼 메뉴가 뜨고, 원하는 블록 종류를 고를 수 있어요. 마치 '명령어'처럼요.

(실제로 블록을 추가하는 단축키예요)

✏️ 따라해보기 — 슬래시 명령어로 블록 만들기
  1. 어제 만든 페이지를 열어요
  2. 빈 줄에서 키보드로 / 입력 → 메뉴가 떠요!
  3. /h1 입력 → Enter → 큰 제목 블록 생성!
  4. /todo 입력 → Enter → 체크박스 블록 생성!
  5. /callout 입력 → Enter → 이모지 강조 박스 생성!
  6. 블록 왼쪽 ⠿ 아이콘을 잡고 드래그 → 순서 바꾸기!
💡 노션 TIP — /를 입력하고 찾고 싶은 블록 이름을 바로 타이핑해도 돼요. 예: /이미지, /표, /동영상

🎨 Day 4
글자 꾸미기 — 인라인 서식
같은 내용도 정돈되면 다시 열고 싶어져요
왜 꾸며야 할까요?

꾸미지 않은 노션 페이지는 글이 다닥다닥 붙어있어서 읽기 힘들어요.

굵게, 색상, 구분선 세 가지만 써도 "정리된 페이지" 느낌이 확 살아나요.

✏️ 따라해보기 — 글자 꾸미기
  1. 텍스트를 드래그해서 선택
  2. 위에 작은 툴바가 떠요 → B(굵게), I(기울임), U(밑줄) 클릭
  3. 툴바에서 A 아이콘 클릭 → 글자색 또는 배경색 선택
  4. 단축키도 있어요 — Ctrl(Cmd)+B 굵게, Ctrl(Cmd)+I 기울임

커버랑 아이콘이 왜 중요해요?

노션 페이지는 제목만 있으면 조금 밋밋해 보여요. 우리가 아는 그 '메모장'처럼요.

커버 이미지를 달면 마치 잡지 표지처럼 페이지에 분위기가 생겨요.

그리고 사이드바에서도 이모지 아이콘이 보여서 훨씬 찾기 쉬워져요.

✏️ 실습 — 내 첫 페이지에 커버 + 아이콘 달아보기
  1. Day 2에서 만든 페이지 열기
  2. 제목 위에 마우스 올리기 → "커버 추가" 클릭
  3. 노션 기본 제공 이미지 중 마음에 드는 것 선택 (직접 업로드도 가능해요!)
  4. 제목 왼쪽 빈 영역 클릭 → "아이콘 추가" → 이모지 선택
  5. 완성! 사이드바에서 이모지가 페이지 옆에 보이는지 확인

DAY 5 · 페이지 구조화16 / 26
📁 Day 5
페이지 안의 페이지

컴퓨터 폴더 VS 노션 페이지

컴퓨터에서는 폴더 안에 파일을 담아요.

노션에는 따로 폴더가 없어요. 대신 페이지 안에 또 다른 페이지를 넣으면 폴더처럼 구조를 잡을 수 있어요.

컴퓨터 폴더가 파일을 담는 것처럼, 노션 페이지는 다른 페이지를 담을 수 있어요.

📂
컴퓨터 폴더
폴더 안에 파일을 담아요. 파일과 폴더가 별개예요.
N
노션 페이지
페이지 자체가 폴더예요. 안에 또 다른 페이지를 넣으면 돼요.
✏️ 따라해보기 — 서브 페이지 만들기
  1. 새 페이지 만들고 제목: "내 대학생활" (나에게 맞는 이름으로)
  2. 페이지 안에서 /page 입력 → Enter
  3. 서브 페이지 이름: "수업 노트" 입력
  4. 같은 방식으로 "과제", "동아리" 서브 페이지 2개 더 만들기
  5. 사이드바에서 들여쓰기된 페이지들 확인!

처음엔 2~3개 큰 카테고리만 만들어도 충분해요. 나중에 언제든 추가하고 바꿀 수 있어요.

DAY 5 · 멘션으로 연결하기18 / 26
📁 Day 5
페이지끼리 연결하기 — 멘션
@를 입력하면 다른 페이지로 바로 이동하는 링크를 만들 수 있어요
멘션이 뭔가요?

인스타그램에서 @누군가 하면 그 사람 프로필로 가죠?

노션에서도 @를 입력하면 다른 페이지로 이동하는 링크가 생겨요.

페이지에서 자주 쓰는 페이지를 멘션으로 연결해두면 바로바로 이동할 수 있어요. 보통 메인페이지에 서브페이지들을 연결시켜놓고 바로 이동하는 용도로 많이 사용되요.

🛠️ 따라해보기 — 멘션 링크 만들기
  1. 메인 페이지("내 대학생활")를 열기
  2. 빈 줄에서 @ 입력 → 검색창 등장
  3. "수업 노트" 검색 → 선택하면 링크 블록 생성!
  4. 링크 클릭 → 해당 페이지로 바로 이동 확인!
@ 입력 후 검색창 + 멘션 링크 화면
💡 여기까지 하면 "내 공간이 생겼다"는 느낌이 와요.

홈 페이지가 왜 필요해요?

노션을 켤 때마다 "어디서부터 시작하지?" 하면서 결국 안 쓰게 되더라고요.

홈 페이지를 만들어두면 노션을 열자마자 오늘 할 일이 보이고, 자주 쓰는 페이지로 바로 이동가능하게 만들 수 있어요. -- 네비게이션 역할

내 노션의 현관문 같은 거예요.

홈 페이지에 넣으면 좋은 것들

  • 나를 소개하는 한 줄 + 이모지 (인생 모토)
  • 오늘의 한마디 or 목표 한 줄 (/callout으로 정돈시키기)
  • 오늘 할 일 체크리스트 (/todo)
  • Day 5에서 만든 하위 페이지들 — 멘션 링크(@)로 홈에 모아두기
    • 💡 노션 TIP — 홈 페이지를 즐겨찾기에 추가하면 사이드바 맨 위에 고정돼요.
    페이지 오른쪽 상단 위 … 클릭 → "즐겨찾기에 추가"!
